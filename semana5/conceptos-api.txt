¿Qué hace esta ruta?
La ruta POST /reporte recibe información enviada desde un cliente
(por ejemplo, una aplicación web o herramientas como Thunder Client
o Postman). Toma el campo "mensaje" del cuerpo de la solicitud y
responde en formato JSON con un estado ("Reporte recibido") y el
mismo mensaje enviado.
 
¿Para qué podría servir en una aplicación comunitaria?
Podría servir para que las personas de una vereda o resguardo envíen
reportes sobre situaciones del territorio, como daños en vías, alertas
ambientales, problemas con el agua o solicitudes a la junta de acción
comunal. Toda esa información quedaría registrada de forma organizada
y accesible para las autoridades comunitarias.
